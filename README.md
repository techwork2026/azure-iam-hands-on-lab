# Azure IAM Hands-On Lab Portfolio

Hands-on Identity and Access Management (IAM) practice using Microsoft Entra ID.

This repository demonstrates real-world IAM tasks including user lifecycle management, Role-Based Access Control (RBAC), Multi-Factor Authentication (MFA), and troubleshooting scenarios.

---

## 🎯 Objectives

- Build real-world IAM experience
- Practice user and access management
- Develop troubleshooting and investigation skills
- Create a job-ready IAM portfolio

---

## 🧠 Skills Demonstrated

- User lifecycle management (create, disable, delete, restore)
- Role-Based Access Control (RBAC)
- Group-based access management
- Multi-Factor Authentication (MFA)
- Conditional Access awareness
- Password reset workflows
- Account troubleshooting
- Access issue resolution
- Sign-in log analysis
- Security investigation

---

### ✅ Day 4 — IAM Troubleshooting

### ✅ Day 5 — RBAC Access Denied Scenario
